Constructor Pattern

Object constructors are used to create specific types of objects - both preparing the object for use and accepting arguments which a constructor can use to set the values of member properties and methods when the object is first created.

Example in JavaScript:

let newObject = {};

let newObject = Object.create( Object.prototype );

let newObject = new Object();

Module Pattern

Modules are an integral piece of any robust application's architecture and typically help in keeping the units of code for a project both cleanly separated and organized.

The Module pattern is a way to provide both private and public encapsulation for classes in conventional software engineering.

Revealing Module Pattern

Definition of functions & variables in the private scope to be returned as an anonymous object with pointers to the private functionality to reveal as public.

Singleton Pattern

In software engineering a singleton is to restrict the instantiation of a class to a single object.

Example in JavaScript:

Config settings or an object that has an API key.
